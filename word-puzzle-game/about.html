📖 About CSE Word Scramble Pro

🎯 Purpose of the Game

CSE Word Scramble Pro is an educational puzzle game designed specifically for Computer Science Engineering students and enthusiasts. The game helps players master technical vocabulary across various CS domains through an engaging word unscrambling format.

Whether you're preparing for exams, interviews, or just want to expand your technical knowledge, this game provides a fun and interactive way to learn and reinforce important CS concepts.

🖥️ Technologies Used

This game is built using pure web technologies:

HTML5 CSS3 Vanilla JavaScript (ES6+) LocalStorage API Google Fonts

📚 Categories Covered

The game includes comprehensive coverage of major Computer Science domains:

  • Programming - Programming languages, paradigms, and concepts
  • Data Structures - Arrays, trees, graphs, linked lists, and more
  • Database - DBMS concepts, SQL, and data modeling
  • Operating Systems - Process management, memory, and scheduling
  • Networking - Protocols, topologies, and network concepts
  • Software Engineering - Development methodologies and practices
  • Computer Fundamentals - Basic hardware and software concepts

📊 Understanding Performance Metrics

The game calculates several advanced performance metrics to help you track your progress:

🎯 Accuracy Percentage

The percentage of correctly answered questions out of total questions attempted.

Accuracy = (Correct Answers / Total Questions) × 100

⚡ PPM (Pieces Per Minute) - Speed-Accuracy Rate

Measures how quickly you can solve puzzles while maintaining accuracy. Higher PPM indicates better performance.

PPM = Correct Answers / Total Time (in minutes)

⏱️ Time Penalty Method

Adjusted time calculation that penalizes wrong answers. This helps measure true efficiency.

Penalty Time = Total Time + (Wrong Answers × 10 seconds)

📈 Success Rate / Efficiency

The ratio of correct answers to total attempts, showing how often you get questions right.

Success Rate = (Correct Answers / Total Attempts) × 100

🎮 How to Play

  • Select a category from the 7 available CS domains
  • Choose a difficulty level (Easy, Medium, Hard)
  • Read the question carefully
  • Unscramble the jumbled word
  • Use hints if needed (but they'll affect your score!)
  • Answer 10 questions to complete the game
  • Wrong answer or timeout ends the game immediately

🏆 Scoring System

Your performance is tracked using localStorage. The leaderboard stores your best scores, accuracy, and PPM ratings. Challenge yourself to reach the top of the leaderboard!

💡 Tips for Success

  • Start with Easy mode to build confidence
  • Use hints sparingly - they're there to help, not rely on
  • Practice all categories to become a well-rounded CS professional
  • Focus on speed AND accuracy - both affect your PPM score
  • Learn from mistakes - wrong answers show you the correct term

Good luck and happy learning! 🚀
